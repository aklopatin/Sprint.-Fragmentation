# Визуализации для отчёта

Сюда сохраняют графики и скриншоты для отчёта:

- графики обучения (loss, mDice, mIoU) — например `training_curves.png`;
- визуализации топ-5 и худших 5 по Dice (исходное изображение, сегментация, наложение маски);
- другие рисунки и скриншоты по необходимости.

Можно использовать произвольную вложенность папок.
