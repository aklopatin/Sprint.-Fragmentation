# practicum_work

Код и промежуточные результаты практикума по семантической сегментации (форк mmsegmentation).

## Структура

- **src/data** — вспомогательный код для этапа 1: EDA, статистики датасета, проверка формата.
- **src/analysis** — скрипты анализа качества: парсинг логов, метрики, лучшие/худшие предсказания, экспорт отчёта.
- **supplementary/viz** — визуализации для отчёта (графики, скриншоты, отрисовки семплов).

Подробная документация кода — в корневом **README.md**, раздел «Этап 5. Документация кода».
